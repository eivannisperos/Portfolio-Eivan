SNEAKR

JANUARY 2016 - APRIL 2016

SNEAKR IS A WEB APPLICATION THAT SERVED AS A DATABASE FOR SNEAKERS. IT WAS INSPIRED BY BOTH MY DESIRE TO LEARN AND TO APPLY DATABASE TECHNOLOGIES USING MYSQL AND PHP AND PREDICTIVE TEXT USING AJAX, AS WELL AS MY OBSESSION WITH URBAN SNEAKER CULTURE (THE YEEZY CRAZE PEAKED DURING THE DURATION OF THE PROJECT).
